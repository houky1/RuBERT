### Проект создан для демонстрации работы с LLM (Large Language Models) и обработки запросов на естественном языке.
Работа выполнена для выпускной работы по цифровой кафедре в рамках курса "ИИ и анализ данных"
Выполнен в среде Google Colab.
Обучающий датасет был сгенерирован синтетически с помощью Sonnet 5 Claude в связи с отсутствием открытых датасетов для обработки обращений клиентов на естественном языке.

01_dataset_analysis.ipynb - анализ датасета для обучения моделей.
02_prepare_dataset.ipynb - подготовка данных для обучения моделей (кодировка признаков, разделение на выборки, токенизация).
03_train_rubert.ipynb - обучение модели Rubert для обработки русского текста.
04_inference.ipynb - Создание функций для определения признаков текстового сообщения.
05_llm.ipynb - демонстрация работы LLM на основе Groq и использование обученных моделей для генерации ответов на естественном языке.
В дальнейшем рекомендуется улучшение структуры проекта, обучение одной модели RuBERT на всех признаках сразу (CLS),
улучшение качества синтетического датасета. Создание единого API для обработки запросов на естественном языке с использованием LLM и обученных моделей.
